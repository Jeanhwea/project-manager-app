* 简介

=pma= 是一个用 Rust 编写的命令行工具，用于批量管理 Git 仓库。

主要功能:
- =release= (=re=): 语义化版本发布，自动编辑版本号、提交、打标签、推送
- =sync= (=s=): 批量同步目录下的 Git 仓库 (pull + push)
- =doctor=: 仓库健康检查，自动修正 remote 命名，可选 =git gc=
- =fork=: 从模板项目 fork 新项目，支持 =.pma.json= 初始化动作
- =snap=: 项目快照，增量 Git 提交

* 安装

通过 Homebrew 安装 (macOS/Linux):

#+BEGIN_SRC bash
brew tap Jeanhwea/tap
brew install pma
#+END_SRC

通过 npm 安装:

#+BEGIN_SRC bash
npm install -g @jeansoft/pma
#+END_SRC

通过源码编译:

#+BEGIN_SRC bash
cargo build --release
#+END_SRC

编译产物位于 =target/release/pma= ，或者直接在 github release 页面下载二进制文件。

* 用法

#+BEGIN_SRC bash
pma release patch                     # 版本发布: v0.1.0 -> v0.1.1
pma release minor                     # 版本发布: v0.1.1 -> v0.2.0
pma release major                     # 版本发布: v0.2.0 -> v1.0.0

# 简写
pma re                                # pma release patch
pma re mi                             # pma release minor
pma re ma                             # pma release major

pma sync                              # 同步当前目录下的仓库
pma sync /path/to/repos -m 5 -a      # 指定目录，深度 5，拉取所有分支
pma doctor --gc                       # 健康检查并执行 git gc
pma fork /path/to/template my-project # 从模板 fork 新项目
pma snap                              # 快照当前目录
#+END_SRC
